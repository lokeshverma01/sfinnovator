Discount Approval App

Shipped

Multi-tier Opportunity discount approvals built with Flow, approval processes, and a manager dashboard — fully declarative, bulk-safe, and testable in a scratch org.

▶ Use it live ✓ Validate (test org + steps) 👁 GitHub repo 📝 Related blog post

The problem

Sales reps were applying steep discounts with no guardrails. The business needed tiered manager approval above 20%, VP approval above 40%, and a clean audit trail — without custom code to maintain.

What I built

A declarative approval solution with three moving parts:

  • A record-triggered Flow that stamps the discount tier.
  • A two-step approval process routing to manager then VP.
  • A manager dashboard with pending/approved/rejected KPIs.
[ screenshot / GIF of the flow in action ]

How it works

The Flow assigns the tier on save; the approval process branches on it:

// Entry criteria (approval step 2 — VP)
Opportunity.Discount_Percent__c > 40

Try it yourself

Spin up the scratch org from the repo, run the install script, and follow the 5-step validation guide to create a 45% discount and watch it route to VP approval.
