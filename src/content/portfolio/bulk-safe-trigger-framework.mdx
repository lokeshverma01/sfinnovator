## Why a framework

Raw triggers grow messy fast. This handler pattern enforces one trigger per object,
keeps logic in testable classes, and makes bulk-safety the default.

## What's inside

- A `TriggerHandler` base class with context virtual methods.
- A dispatcher that routes trigger events to the right handler.
- Examples and tests.
