## The problem

Sales reps applied steep discounts with no guardrails. The business needed tiered
approval — manager above 20%, VP above 40% — with a clean audit trail and no custom
code to maintain.

## What I built

- A record-triggered Flow that stamps the discount tier on save.
- A two-step approval process routing to manager, then VP.
- A manager dashboard with pending / approved / rejected KPIs.

## Try it yourself

Spin up the scratch org from the repo, run the install script, then follow the
5-step validation guide to create a 45% discount and watch it route to VP approval.
