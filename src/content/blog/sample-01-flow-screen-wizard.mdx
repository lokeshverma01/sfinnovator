## Plan the screens

Sketch the steps before building: welcome, details, conditional branch, review, confirm.

## Build it

1. New **Screen Flow** in Flow Builder.
2. Add Screen elements for each step.
3. Use a **Decision** element to branch.

```apex
// Optional invocable to post-process the record
public class IntakeRouter {
    @InvocableMethod
    public static void route(List<Id> recordIds) { /* ... */ }
}
```
