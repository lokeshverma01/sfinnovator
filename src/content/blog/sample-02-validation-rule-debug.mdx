## What we expected

The rule should only fire when the user changes the amount.

## What actually happened

It fired on every save. Formula fields are recalculated each save, so ISCHANGED is always true.

## Fix

```
AND( ISCHANGED(Amount__c), Amount__c > 10000 )
```
