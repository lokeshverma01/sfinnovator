## Why an approval process

Discount governance is a classic use of Salesforce's built-in **approval processes**.
They lock the record, route it to the right approver, and track every decision — all
declaratively.

## Step 1 — Create the approval process

1. Setup → **Approval Processes** → select **Opportunity**.
2. Use the **Jump Start Wizard** for a single-step approval.
3. Name it `Discount Over 20%`.

## Step 2 — Entry criteria

Set the criteria so only high discounts enter the process:

```
Opportunity: Discount Percent  greater than  20
```

## Step 3 — Approver and actions

- **Approver:** _Automatically assign to the submitter's manager._
- **On submit:** lock the record and set a `Discount Status` field to `Pending`.
- **On approval:** unlock and set `Discount Status` to `Approved`.
- **On rejection:** unlock and set `Discount Status` to `Rejected`.

## Step 4 — Test it

Create an Opportunity with a 25% discount and click **Submit for Approval**. It should
lock and notify the manager. Approve it as the manager and confirm it unlocks.

## Notes

Keep the process single-step to start; you can add escalation or multiple approvers
later without rebuilding it.
