## When to use it

Backend integrations with no user present. You sign a JWT with your private key; Salesforce
trusts the matching certificate.

```bash
curl https://login.salesforce.com/services/oauth2/token \
  -d grant_type=urn:ietf:params:oauth:grant-type:jwt-bearer \
  -d assertion=$SIGNED_JWT
```

## Notes

Keep the private key in a secret store; never commit it.
