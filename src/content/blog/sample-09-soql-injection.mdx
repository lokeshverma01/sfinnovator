## Vulnerable

```apex
String q = 'SELECT Id FROM Account WHERE Name = \'' + userInput + '\'';
List<Account> a = Database.query(q);
```

## Safe

```apex
List<Account> a = [SELECT Id FROM Account WHERE Name = :userInput];
```
